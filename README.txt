Quick deploy on GitHub Pages:
1. Create a new GitHub repo (e.g. rosdj-site).
2. Upload all files from this folder to the repo root (index.html + images).
3. In GitHub repo Settings → Pages choose Branch: main and Folder: / (root). Save.
4. In GitHub Pages -> Custom domain add: rosdj.es
5. In IONOS DNS settings add A records pointing to GitHub Pages IPs and CNAME for www (see GitHub Pages docs).
