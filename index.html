Camelot Wheel

The Camelot Wheel is essential for harmonic mixing in DJ sets. It helps DJs mix tracks in the same key or compatible keys, creating smooth transitions and maintaining musical flow throughout the performance.

Key compatibility:
• Same number = Perfect mix
• ±1 number = Energy change
• Same letter = Relative minor/major
